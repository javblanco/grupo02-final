DESCRIPCIÓN
-----------

Anexo es una aplicación ESB que permite al usuario la gestión de las conexiones, los flujos
y las instancias creadas, brindando al usuario una capa de abstracción sobre el envió de datos.

La aplicación permite registrar, borrar y eliminar dichas conexiones, flujos e instancias, y
realizar una trazabilidad.


ESTADO DEL PROYECTO
-------------------

Anexo permite al usuario la creación, edición y eliminación de Flujos, instancias y conexiones,
respetando las relaciones establecidas sobre estas.

BUGS CONOCIDOS
--------------

la eliminación de esta (Falta añadir modal avisando)

*Los Date están en orden ingles, falta conversión a modelo Europeo de forma automatizada.
